# Тесты Morning Quiz Bot

## Описание

Эта папка содержит тесты для проверки работоспособности системы Morning Quiz Bot.

## Запуск тестов

### Запуск всех тестов (рекомендуется)
```bash
python tests/run_all_tests.py
```

### Отдельные тесты

#### Тест целостности системы
```bash
python tests/system_integrity_test.py
```

#### Тест устойчивости системы (защита от дурака)
```bash
python tests/robustness_test.py
```

#### Тест навигации по меню
```bash
python tests/menu_navigation_test.py
```

### Запуск всех тестов вручную
```bash
python tests/system_integrity_test.py
python tests/robustness_test.py
python tests/menu_navigation_test.py
```

## Структура

- `__init__.py` - Инициализация пакета тестов
- `README.md` - Этот файл
- `run_all_tests.py` - Скрипт запуска всех тестов
- `system_integrity_test.py` - Тест целостности системы
- `robustness_test.py` - Тест устойчивости и защиты от дурака
- `menu_navigation_test.py` - Тест навигации по меню и симуляции кнопок
- `TESTING_REPORT.md` - Итоговый отчет по тестированию

## Назначение тестов

### system_integrity_test.py
Проверяет:
- Импорты всех модулей
- Загрузку конфигурации
- Структуру данных
- Загрузку данных
- Импорт обработчиков
- Импорт модулей логики

### robustness_test.py
Проверяет:
- Обработку некорректных данных конфигурации
- Обработку некорректных ID пользователей
- Обработку некорректных ID чатов
- Валидацию структуры вопросов
- Граничные случаи
- Согласованность данных

### menu_navigation_test.py
Проверяет:
- Навигацию по главному меню
- Навигацию по меню викторины
- Навигацию по меню настроек
- Навигацию по меню рейтинга
- Навигацию по меню помощи
- Навигацию по меню резервных копий
- Симуляцию нажатия кнопок меню
- Обработку ошибок в меню

## Важно

Эти тесты предназначены для проверки работоспособности системы **БЕЗ ЗАПУСКА БОТА**.
Они проверяют только импорты, конфигурацию, структуру данных и функциональность модулей.

## Добавление новых тестов

При создании новых тестов:
1. Добавьте их в папку `tests/`
2. Обновите этот README
3. Добавьте тест в `run_all_tests.py`
4. Убедитесь, что тесты не запускают бота
5. Используйте описательные имена файлов
6. Добавьте тест в итоговый отчет

## Особенности теста навигации по меню

Тест `menu_navigation_test.py` специально разработан для проверки:
- **Симуляции нажатия кнопок** - имитирует действия пользователя
- **Навигации по меню** - проверяет все разделы интерфейса
- **Обработки ошибок** - тестирует устойчивость к некорректным данным
- **Структуры меню** - проверяет корректность команд и настроек

Этот тест особенно важен для проверки пользовательского интерфейса и навигации.
